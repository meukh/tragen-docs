class tragen.client_tragen.tragenClient.ItemValueReader(items_list=[], period=0.5)[source]¶

Bases: threading.Thread

This class represents the reader thread that randomly reads a variable’s or a property’s value in the server. By default, every client of a Tragen context has one. It is created at initialization time of Tragen context.

Parameters:
  • items_list – list of the server’s readable items (e.g variable values)
  • period – frequency by which the reader is sending requests the server.

class tragen.server_tragen.tragenServer.AsynchronousNotfication(event, var, var_name, lower_limit=-inf, upper_limit=inf, auto=False)[source]¶

Bases: threading.Thread

This class represents the notifier thread that periodically checks wether the value of a variable has reached its critical value or not, it will report/notify the subscribed client accordingly through an event. By default, every updatable variable of a Tragen context, that changes periodically, has one. It is created at initialization time.

Parameters:
  • event – Event object
  • var – The server’s variable node
  • var_name –
  • lower_limit – Lower critical value
  • upper_limit – Upper critical value
  • auto – If set, then critical values will be set automatically.

class tragen.server_tragen.tragenServer.VariableNodeValue(var, regular_update=True, period=0.1, val_stddev=0.15, pd_stddev=0.6)[source]¶

Bases: threading.Thread

This class represents the updater thread that randomly updates a variable’s value with regular or irregular intervals. By default, every updatable variable of a Tragen context has one. It is created at initialization time of the context.

Parameters:
  • var – The server’s variable node
  • regular_update – Boolean. If true the updates will be done at fixed intervals.
  • period – Interval between updates.
  • val_stddev – The standard deviation of the variable’s value changes.
  • pd_stddev – The standard deviation of the interval changes. Used only if regular_update is not set.
